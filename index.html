List Operations


val nums = List(1, 2, 3, 4)

val printer = (x: Int) => println(x)
val incrementor = (x: Int) => x + 1

nums.foreach( num => printer(num) )
// prints the numbers 1-5

nums.map( num => incrementor(num) )
// => List[Int] = List(2, 3, 4, 5)

Underscore

Function Composition

\[ (f\circ g)(x) \]

Just like Algebra

Functions are nothing more than a transformation of an input.

Function Composition in Scala

\[ f(x) = {x \over 2},~ g(x) = x^2 \rightarrow (f\circ g)(x) \] 

val f: Int => Double = x => x / 2.0
  // => f: Int => Double = <function1>
val g: Int => Int = x => x * x
  // => g: Int => Int =  <function1>
        

val x = 5
f(g(x))
  // => Double = 12.5
        

val fog = f compose g
  // => Int => Double = <function1>
fog(x)
  // => Double = 12.5
        

g andThen f
  // => Int => Double = <function1>
